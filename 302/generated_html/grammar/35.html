X こそ:
It is very (indeed) X that

Explanation: This particle こそ expresses that of all possible candidates, meaning "It is indeed X that fits the description [in the predicate]." The best example of this is こちらこそ to express "thank you," indicating "It is indeed me that should thank you."

例. 今日こそこの仕事を仕上げるぞ。
Today, for sure, I'll get this work done.
